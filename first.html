Keyboard Shortcuts

These are the shortcuts for the Mac programs we use in Mobile Apps class.

⌘-R Refresh Safari

⌘-C Copy Safari / Windows
⌘-V Paste Safari / Windows
⌘-A Select All Safari / Windows
⌘-Z Undo Safari / Windows
⌘-F Find Safari / Windows
⌘-Shift-T Bring Back Tab Safari / Windows
